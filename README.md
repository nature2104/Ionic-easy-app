# Ionic-easy-app

> A Ionic3.X project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8100
ionic serve

# build for production with minification
ionic build --prod

```

For a detailed explanation on how things work, check out the [guide](https://ionicframework.com/docs/).
